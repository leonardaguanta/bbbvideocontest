== Description ==
Easy way to display the number of post in that particular tag by selecting from admin dashboard widget.

Through this plugin user can,
* Add the custom text as a title to display
* Able to select the specific tag, so that that tag post title will be displayed in webpage.
* Able to select the number of post title can be display in the webpage, ranges from 1 to 10.
* Show/Hide Bullet style
* Restrict number of characters in title

For More info - http://buffercode.com/tag-widget-wordpress-plugin/

== Installation ==

1. Upload the "tag-widget" directory to the plugins directory.
2. Go to the plugins setting page and activate "tag Widget"
3. Go to Appearance --> Widget --> Drag the Tag Widget and place it in appropriate widget location and configure it as per your requirement
